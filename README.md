## Setup
- New Folder static\heatmaps


- PYTHON ENVIRONMENT
```
# สร้าง virtual environment
python -m venv .venv

# virtual environment ver.3.13
py -3.13 -m venv .venv

# เปิดใช้งาน
.\.venv\Scripts\activate      # WINDOWS
source .venv/bin/activate     # MAC/LINUX

# ติดตั้ง pip packages
pip install -r requirements.txt
```
- Node.js packages
```
npm install
```

## Usage
- Run Node.js server
```
node app.js
```
